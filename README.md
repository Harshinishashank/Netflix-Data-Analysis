# Netflix Global Content Analysis
An exploratory data analysis (EDA) of the Netflix library using Python, Pandas, and Seaborn.

## 📊 Key Insights
* **Global Leader:** The United States and India are the top content producers.
* **Content Split:** Analyzed the ratio of Movies vs. TV Shows across different regions.
* **Growth Trend:** Visualized the explosion of Netflix original content from 2015-2021.

## 🛠️ Tools Used
* **Python** (Data Processing)
* **Pandas & NumPy** (Data Cleaning and Normalization)
* **Matplotlib & Seaborn** (Data Visualization)
* **Google Colab** (Development Environment)

## 🚀 How to Run
1. Open the `.ipynb` file in Google Colab.
2. Add your Kaggle API key to the `userdata` secrets.
3. Run all cells to reproduce the analysis.
